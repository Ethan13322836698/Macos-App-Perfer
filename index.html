一、开发工具篇

1. Homebrew

Mac 的优秀包管理器，一行命令解决应用安装，不需要手动下载。

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

前往官网

2. Ollama

本地离线运行 LLM（大语言模型）。

brew install --cask ollama

前往官网

3. Sublime Text

强大、简洁的文本编辑器，编程体验优秀。

brew install --cask sublime-text

前往官网

4. Ghostty

比 macOS 自带 Terminal 更快流畅，支持 GPU 加速、连字渲染、分屏。

brew install ghostty

前往官网
